Detailed Description

file containing functions meant to disassemble the code from an elf script

Author
Luciano Mantovani
Date
November 2014 This file has functions called by the main program when it has to disassemble an elf script.

for now, it has two functions. One that loads all the code operations into a lookup table, and another that recieves as a parameter a 32 bit word and returns the operation given by the word.
