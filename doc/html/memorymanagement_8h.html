struct ptype* creatememory ( struct ptype *  mips,
int  size 
)

creates a block of memory

uses a structure ptype and an integer size to create a new block. this block will have a size mips->size and will start from memory position mips->memrealpointbase.

sets up a segment's parameters

this function does not literally create a segment inside a memory block. Instead, it loads the information concerning a created segment into a lookup table, which will be referenced when reading from or writing to a segment.
