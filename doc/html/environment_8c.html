Detailed Description

file containing functions to run the environment for the user

Author
Luciano Mantovani
Date
October 2014 this file has functions that react to the lines entered by the user. The commands from the file environmentcommands.c are called by functions from this file.

Function Documentation

struct ptype* analize ( struct ptype *  mips)

this function is a distributer, it calls a function from file environmentcommands.c according to the command entered in the entry

the only command that doesn't call any function is exit

struct ptype* parseentry ( struct ptype *  mips)

this function parses an entry in the environment

it parses the entire line entered and stores the command in the field char *command from the struct ptype, and stores all the arguments in the field char *argenv[] from the structure ptype

void restart ( struct ptype *  mips)

this function sets to null the values in command and in argenv array

it just to reset and prepare the variables for the next command

void runenv ( struct ptype *  mips)

this function runs the environment program and allows the user to enter commands

after obtaining a line from stdin, it parses it and analizes the command. The function analize() also executes it, and if there's no report (mips->report = 0) the program leaves.

Note
this function is thought to be called within an endless loop
